Змейка с ИИ-ботом (Snake AI Bot)
Автоматическая игра "Змейка", где вместо игрока управляет искусственный интеллект.

Описание проекта
Это реализация классической игры "Змейка", в которой управление полностью автоматизировано. ИИ-бот самостоятельно собирает яблоки, избегает столкновений со стенами и собственным телом, используя продвинутые алгоритмы поиска пути.

Особенности
Полностью автоматический бот: Змейка управляется искусственным интеллектом

Улучшенный алгоритм ИИ: Использует алгоритм A* для поиска оптимального пути

Минималистичный дизайн: Черный фон, простые элементы

Правильная цветовая схема:

Змейка состоит из двух  оттенков зеленого
Яблоки - красные квадраты (#ff0000)

Плотные блоки: Отсутствие прорезей между сегментами змейки

Адаптивная сложность: Скорость игры увеличивается по мере набора очков

Технические особенности
Алгоритм поиска пути: A* с манхэттенской эвристикой

Многоуровневая стратегия: Приоритетное следование вычисленному пути, резервные стратегии при невозможности найти путь

Избегание тупиков: Оценка свободного пространства для предотвращения застревания

Оптимизация производительности: Пересчет пути происходит раз в 3 хода

Пробел: Пауза/Продолжение

Как запустить
Просто откройте файл index.html в любом современном браузере. Игра начнется автоматически.

Технологии
HTML5 Canvas для отрисовки

Чистый JavaScript (без зависимостей)

LocalStorage для сохранения рекорда

Структура кода
initGame(): Инициализация игрового состояния

calculatePathToApple(): Поиск пути к яблоку с помощью A*

calculateBotDirection(): Выбор направления движения

drawGame(): Отрисовка игрового поля

updateGame(): Обновление игровой логики

Особенности ИИ
Алгоритм A*: Основной метод поиска пути к яблоку

Резервные стратегии: Если путь не найден, использует эвристические методы

Оценка безопасности: Проверка столкновений с границами и телом

Избегание тупиков: Предпочтение клеток с большим количеством выходов
